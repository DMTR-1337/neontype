# NeonType

A minimal, WPM (Words Per Minute) test built with JavaScript.

---

### Features
* **Real-time Stats:** Live WPM and Accuracy tracking.
* **Zero Lag:** Instant response time.
* **Auto-Focus:** Click anywhere to start typing.

### Quick Start
1. Clone the repo.
2. Open `index.html` in your browser.
3. Start typing. Use `Space` to submit words.

### Tech
* Vanilla ES6 JS
* CSS3 + TailwindCSS
